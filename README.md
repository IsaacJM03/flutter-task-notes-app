# flutter-task-notes-app

A Flutter application repository containing the Task Notes Manager app - a simple and elegant solution for managing task notes with priorities.

## Project: task_notes_manager

The main application is located in the `task_notes_manager/` directory. This Flutter app allows users to manage their tasks with title, description, and priority fields.

### Features
- Home screen with dynamic task list
- Add task form screen with validation
- Priority-based color coding (High, Medium, Low)
- Clean Material Design 3 UI
- Navigation using Flutter Navigator

See the [task_notes_manager README](task_notes_manager/README.md) for detailed information about the app.
